<h1>🧠 Bank Customer Churn Analysis Lab</h1>

<p>
This repository contains a machine learning classification activity developed during the 
<strong>Oracle Next Education Bootcamp (Oracle + Alura)</strong>.
</p>

<p>
The objective of this exercise is to analyze customer data and build models capable of predicting 
<strong>customer churn</strong> (when a client stops using the bank's services).
</p>

<hr>

<h2>📊 Project Overview</h2>

<p>
The analysis follows a typical Machine Learning workflow:
</p>

<ul>
<li>Exploratory Data Analysis (EDA)</li>
<li>Data preprocessing and feature encoding (One-Hot Encoding / Dummy variables)</li>
<li>Training and evaluation of classification models</li>
<li>Prediction for a new customer example</li>
</ul>

<hr>

<h2>🤖 Machine Learning Models Tested</h2>

<p>The following classification algorithms were tested:</p>

<ul>
<li>🌳 Decision Tree</li>
<li>📏 K-Nearest Neighbors (KNN)</li>
<li>🧩 Dummy Classifier (baseline model)</li>
</ul>

<p>
The <strong>Dummy Classifier</strong> was used as a baseline to compare whether the real models were 
actually learning patterns from the data.
</p>
